# Netflix Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Data Cleaning
- Descriptive Statistics
- Trend Analysis
- Data Visualization

## Key Insights
- Movies are more common than TV Shows.
- The United States contributes the most content.
- TV-MA is one of the most frequent ratings.
